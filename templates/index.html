Step 1: Upload CSV File

Upload a csv file containing the marks in the following format.

Name, Subject 1, Subject 2, Subject 3, Subject 4, Subject 5


ℹ️ You can have any number of subjects in the file.


If you have an excel sheet with the name of the students and their marks, select File -> Save As and choose Comma Separated Value (.csv) as the file extension and save the file to get the csv file.

Step 2: Download Graphs

Wait for the application to finish generating the graphs and download the zip file containing graphs for each student.
